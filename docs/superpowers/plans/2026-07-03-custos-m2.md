# Custos M2 Implementation Plan — Synth GUI in a Custos-owned Floating Window

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give Custos a minimal own editor (intercepting the host's generic 5000-slider view) with a Show/Hide button that opens the inner synth's original GUI in a Custos-owned floating top-level window.

**Architecture:** A `SynthWindow` (top-level `juce::DocumentWindow`) hosts the inner synth's `AudioProcessorEditor` (from `createEditorAndMakeActive()`). The window is owned by `CustosProcessor` (not by the editor) so it survives the host closing the plugin panel. `CustosEditor` is a thin view whose button calls `processor.toggleSynthWindow()`. Hide == Close == destroy the window + hosted view; Show == create it fresh; `synthWindow == nullptr` is the single source of truth for visibility.

**Tech Stack:** C++17, CMake, JUCE 8.0.14 (`juce_gui_basics` for the editor/window), Catch2 v3.15.1, VST3 (Windows).

## Global Constraints

- **Language/std:** C++17. **JUCE:** 8.0.14. **Catch2:** v3.15.1.
- **Window ownership:** the `SynthWindow` is owned by `CustosProcessor`, never by `CustosEditor`.
- **Visibility semantics:** Hide == Close == destroy the window and its hosted view (synth keeps playing); Show == create the window fresh; `isSynthWindowVisible() == (synthWindow != nullptr)`.
- **One `SynthWindow` per Custos instance** (the inner `IPlugView` attaches to only one parent).
- **Destruction order (critical):** in `~CustosProcessor`, destroy `synthWindow` **before** `inner`.
- **Window behaviour:** normal, closable, **not** always-on-top; first-open position is OS default (native title bar). No docking/OSC in M2.
- **Threading:** all window create/show/hide/destroy on the message thread; the audio thread never touches `synthWindow`.
- **`hasEditor()` becomes `true`** (was `false` in M1); `createEditor()` returns a `CustosEditor`.
- **Do not change** the M1 facade/passthrough/loader/host-trace behaviour. The parameter facade stays exactly `kFacadeParamCount` with the same ids.
- **Build the `Custos_VST3` target** (not `Custos`) to produce the plugin bundle; unit tests run headless and must construct GUI objects under a `juce::ScopedJuceInitialiser_GUI`.

- [ ] **Step 1: Write src/SynthWindow.h**

```cpp
#pragma once
#include <juce_gui_basics/juce_gui_basics.h>
#include <functional>

namespace custos
{
// A Custos-owned top-level window that hosts a hosted plugin's editor component (takes ownership
// of it via setContentOwned). Closing it (title bar) invokes onClose asynchronously; the owner
// (CustosProcessor) uses that to destroy this window. The hosted synth itself is unaffected.
class SynthWindow : public juce::DocumentWindow
{
public:
    SynthWindow (const juce::String& title, juce::Component* editor, std::function<void()> onClose);

    void closeButtonPressed() override;

private:
    std::function<void()> onCloseCallback;

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR (SynthWindow)
};
}
```

- [ ] **Step 2: Write src/SynthWindow.cpp**

```cpp
#include "SynthWindow.h"

namespace custos
{
SynthWindow::SynthWindow (const juce::String& title, juce::Component* editor, std::function<void()> onClose)
    : juce::DocumentWindow (title, juce::Colour (0xff2b2b2b), juce::DocumentWindow::closeButton),
      onCloseCallback (std::move (onClose))
{
    setUsingNativeTitleBar (true);
    setContentOwned (editor, true);          // takes ownership; sizes the window to the editor
    setResizable (false, false);             // synth editors drive their own size; window follows
    centreWithSize (getWidth(), getHeight());
    setVisible (true);
}

void SynthWindow::closeButtonPressed()
{
    // Do not delete ourselves inside our own callback; defer to the owner.
    if (onCloseCallback)
        juce::MessageManager::callAsync (onCloseCallback);
}
}
```

juce::String CustosProcessor::innerSynthName() const
{
    return inner != nullptr ? inner->getName() : juce::String ("(none)");
}

void CustosProcessor::showSynthWindow()
{
    if (inner == nullptr) return;                              // nothing to show
    if (synthWindow != nullptr) { synthWindow->toFront (true); return; }
    if (auto* ed = inner->createEditorAndMakeActive())             // null if the synth has no editor
        synthWindow = std::make_unique<SynthWindow> (
            kProduct + juce::String (" - ") + inner->getName(),
            ed,
            [this] { hideSynthWindow(); });
}

void CustosProcessor::hideSynthWindow()
{
    synthWindow.reset();
}

void CustosProcessor::toggleSynthWindow()
{
    if (isSynthWindowVisible()) hideSynthWindow();
    else                        showSynthWindow();
}
```

#pragma once
#include <juce_audio_processors/juce_audio_processors.h>
#include <juce_gui_basics/juce_gui_basics.h>

namespace custos
{
class CustosProcessor;

// Minimal plugin editor: product name + "Synth: <name>" status + a Show/Hide toggle button that
// opens/closes the inner synth's own GUI in the processor-owned floating window. Deliberately tiny
// so the host shows this instead of its generic 5000-slider parameter view.
class CustosEditor : public juce::AudioProcessorEditor
{
public:
    explicit CustosEditor (CustosProcessor&);
    ~CustosEditor() override;

    void resized() override;
    void paint (juce::Graphics&) override;

private:
    void refresh();   // sync status text + button label from processor state

    CustosProcessor& proc;
    juce::Label      titleLabel;
    juce::Label      statusLabel;
    juce::TextButton synthButton { "Show Synth" };

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR (CustosEditor)
};
}
```

- [ ] **Step 2: Write src/CustosEditor.cpp**

```cpp
#include "CustosEditor.h"
#include "CustosProcessor.h"

namespace custos
{
CustosEditor::CustosEditor (CustosProcessor& p)
    : juce::AudioProcessorEditor (p), proc (p)
{
    titleLabel.setText (kProduct, juce::dontSendNotification);
    titleLabel.setJustificationType (juce::Justification::centredLeft);
    addAndMakeVisible (titleLabel);

    statusLabel.setJustificationType (juce::Justification::centredLeft);
    addAndMakeVisible (statusLabel);

    synthButton.onClick = [this]
    {
        proc.toggleSynthWindow();
        refresh();
    };
    addAndMakeVisible (synthButton);

    refresh();
    setSize (320, 120);
}

CustosEditor::~CustosEditor() = default;

void CustosEditor::refresh()
{
    statusLabel.setText ("Synth: " + proc.innerSynthName(), juce::dontSendNotification);
    synthButton.setButtonText (proc.isSynthWindowVisible() ? "Hide Synth" : "Show Synth");
    synthButton.setEnabled (proc.hasInnerSynth());
}

void CustosEditor::resized()
{
    auto r = getLocalBounds().reduced (12);
    titleLabel.setBounds (r.removeFromTop (28));
    r.removeFromTop (8);
    statusLabel.setBounds (r.removeFromTop (24));
    r.removeFromTop (12);
    synthButton.setBounds (r.removeFromTop (32).removeFromLeft (140));
}

void CustosEditor::paint (juce::Graphics& g)
{
    g.fillAll (juce::Colour (0xff1e1e1e));
}
}
```
