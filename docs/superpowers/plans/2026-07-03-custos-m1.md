# Custos M1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A buildable VST3 instrument (`Custos`) that reports a fixed 5000-parameter facade, loads one hard-coded synth inside itself, passes MIDI→Stereo through it, and mirrors the inner synth's parameters onto facade indices `0..innerCount-1` — plus a host-call trace to run the GP count-timing experiment.

**Architecture:** JUCE/C++ plugin. `CustosProcessor : juce::AudioProcessor` is the VST3. It owns exactly `kFacadeParamCount` (5000) `FacadeParameter` objects (each `HostedAudioProcessorParameter` with a stable per-index VST3 id `custos_<i>`). At construction it loads a hard-coded VST3 synth via `SynthLoader` (a thin wrapper over `juce::AudioPluginFormatManager`) and `InnerBinding` points facade `0..n-1` at the inner synth's parameters; the rest stay inert. `processBlock` forwards MIDI+audio to the inner synth. Core logic lives in a `custos_core` static library shared by the plugin target and a Catch2 test executable, so the facade/binding/passthrough logic is unit-tested against a fake inner processor with no real synth installed.

**Tech Stack:** C++17, CMake (FetchContent), JUCE 8.0.14, Catch2 v3.15.1, VST3 (Windows first).

## Global Constraints

- **Language/std:** C++17 (`CMAKE_CXX_STANDARD 17`, required).
- **CMake:** minimum `3.22`.
- **JUCE:** `8.0.14` (FetchContent `GIT_TAG 8.0.14`).
- **Catch2:** `v3.15.1` (FetchContent `GIT_TAG v3.15.1`).
- **Plugin type:** VST3 **instrument** — `IS_SYNTH TRUE`, `NEEDS_MIDI_INPUT TRUE`, no audio input, **always stereo output**.
- **Branding:** `PRODUCT_NAME "Custos"`, `COMPANY_NAME "Kapellmeister"`, `PLUGIN_MANUFACTURER_CODE Kplm`, `PLUGIN_CODE Cst1`.
- **Facade:** compile-time constant `custos::kFacadeParamCount = 5000`. Facade parameter VST3 ids are stable per index: `"custos_" + index`. Indices `0..innerCount-1` mirror the inner synth 1:1; `innerCount..4999` are inert. **Custos mirrors only — it never maps, scales, or interprets.**
- **Threading (M1):** the inner synth is loaded/attached once at construction; no runtime swap. Plugin loading never happens on the audio thread.
- **No OSC, no GUI, no swap in M1** (later milestones). `hasEditor()` returns `false`.
- **Hard-coded synth path:** compile definition `CUSTOS_HARDCODED_SYNTH_PATH` (set via CMake cache var). Empty ⇒ Custos runs as a silent passthrough with no inner synth (still loads in a host).

namespace custos
{
// Points facade[0..n-1] at the inner processor's parameters (host order) and clears
// facade[n..end] to inert, where n = min(inner param count, facade size).
class InnerBinding
{
public:
    static int  bind (juce::AudioProcessor& inner,
                      const std::vector<FacadeParameter*>& facade) noexcept;
    static void unbindAll (const std::vector<FacadeParameter*>& facade) noexcept;
};
}

- [ ] **Step 2: Write src/InnerBinding.cpp**

```cpp
#include "InnerBinding.h"

namespace custos
{
int InnerBinding::bind (juce::AudioProcessor& inner,
                        const std::vector<FacadeParameter*>& facade) noexcept
{
    const auto& innerParams = inner.getParameters();
    const int n = juce::jmin (innerParams.size(), (int) facade.size());
    for (int i = 0; i < (int) facade.size(); ++i)
        facade[i]->bind (i < n ? innerParams[i] : nullptr);
    return n;
}

void InnerBinding::unbindAll (const std::vector<FacadeParameter*>& facade) noexcept
{
    for (auto* f : facade) f->bind (nullptr);
}
}
```

#include "SynthLoader.h"

namespace custos
{
std::unique_ptr<juce::AudioPluginInstance> SynthLoader::loadVST3 (
    const juce::String& path, double sampleRate, int blockSize, juce::String& errorMessage)
{
    if (path.isEmpty())        { errorMessage = "empty synth path"; return nullptr; }
    if (! juce::File (path).exists()) { errorMessage = "file not found: " + path; return nullptr; }

    juce::AudioPluginFormatManager formatManager;
    formatManager.addDefaultFormats();

    juce::VST3PluginFormat vst3;
    juce::OwnedArray<juce::PluginDescription> descriptions;
    vst3.findAllTypesForFile (descriptions, path);
    if (descriptions.isEmpty()) { errorMessage = "no VST3 types in: " + path; return nullptr; }

    return formatManager.createPluginInstance (*descriptions.getFirst(),
                                               sampleRate, blockSize, errorMessage);
}
}
